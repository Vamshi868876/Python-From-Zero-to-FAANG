# 💼 Interview Questions

## What is a variable?
A variable is a symbolic name that references a value stored in memory.

---

## Is Python statically typed or dynamically typed?
Python is dynamically typed.

---

## Difference between assignment and equality?
Assignment:
`x = 10`

Equality:
`x == 10`

---

## What happens internally when:
```python
name = "Rahul"
```
Python creates a string object, stores it in memory, and binds the variable name to that object.
